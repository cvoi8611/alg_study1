문제 인식

    N-Queen 문제는 크기가 N × N인 체스판 위에 퀸 N개를 서로 공격할 수 없게 놓는 문제이다.

    N이 주어졌을 때, 퀸을 놓는 방법의 수를 구하는 프로그램을 작성하시오.

문제 해결 방식

    퀸이 공격가능한 범위에 퀸이 존재하지 않게 하면서 퀸을 놓는 방법을 작성하면 된다.

    체스판 위, 각각의 줄에는 퀸이 1개만 존재 가능하다
    첫번째 줄에 퀸 위치를 n번째 부터 둔다

    검사해야할 조건은 3가지다
    1) 가로행에 퀸이 존재하는지
    2) 세로행에 퀸이 존재하는지
    3) 대각선에 퀸이 존재하는지

    위 3가지 조건을 모두 만족할 떄, 퀸을 위치할 수 있다
    조건에 만족하지 않으면, n+1 자리에 배치하고 다시 조건을 확인한다 (재귀)

슈도 코드



bool 조건 확인 함수 {
    3가지 조건 만족 시 false 반환
    불만족 시 true 반환
}

퀸 배치 함수(n) {
    for (입력받은 행 갯수만큼 반복){
        if (bool 조건 확인 함수){
            퀸 배치 함수 (n+1)
        }
    }
}

int main() {
    퀸 배치 함수 (0)
}
